Integration Guide
3 lines of code · No build step required
How to embed
1
Add the script
Paste the <script> tag once in your page. No npm, no build step, no framework required.
2
Place the element
Drop <bloomfire-search> anywhere in your HTML. Works in Angular, React, Vue, or plain HTML.
3
Pass your token
Set token and community-id. The widget authenticates silently — no login prompt shown.
Embed code
Copy and paste into any webpage 
<!-- Step 1: Load widget script once per page -->
<script src="https://cdn.bloomfire.com/widget/bloomfire-search.js"></script>

<!-- Step 2: Place the element where you want it -->
<bloomfire-search
  mode="search"
  community-id="YOUR_COMMUNITY_ID"
  token="YOUR_AUTH_TOKEN"
></bloomfire-search>

<!-- Step 3 (optional): Listen to search events -->
<script>
  document.querySelector('bloomfire-search')
    .addEventListener('bloomfire-search', (e) => {
      console.log('Query:', e.detail.query);
      console.log('Results:', e.detail.resultCount);
    });
</script>
Attributes
AttributeValuesRequiredDescription
mode "search" · "synapse" Optional Default is search. search shows semantic results with relevance scores and filters. synapse enables full conversational AI. Users can switch between both modes inside the widget.
community-id String Required Your Bloomfire community identifier. Scopes all results to your community.
token String Required Auth token for the current user. Passed silently — no login prompt shown.
